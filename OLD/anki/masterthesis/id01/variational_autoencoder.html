What is the idea behind variational autoencoders (VAEs)?
VAEs belong to the generative modeling approaches, more concretly, VAEs aim to create a model distribution from which we can sample to generate data that is close to the data used during training.

Taken from this wonderful post.

The basic idea/hope is that since $\textbf{z}$ is a lower-dimensional vector (bottleneck), it should be a compressed and informative representation of the original data $\textbf{x}$.
